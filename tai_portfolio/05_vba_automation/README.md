# VBA Automation – BD Report Macro

## What it does
Automates a weekly Business Development pipeline report in Excel using VBA.

## How to use
1. Open Excel → Press `Alt + F11` to open VBA Editor
2. Go to `Insert → Module`
3. Paste the contents of `BD_Report_Macro.bas`
4. Close VBA Editor
5. Press `Alt + F8` → Select `GenerateBDReport` → Click **Run**

## Output
Generates two sheets:
- `Raw_Data` — proposal data (customer, industry, status, value)
- `BD_Report` — formatted summary with pipeline by status and industry

- Automated report generation
